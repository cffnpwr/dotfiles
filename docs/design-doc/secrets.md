# シークレット機構

リポジトリに含める秘密情報は[age](https://github.com/FiloSottile/age)で暗号化する。

## 暗号化方式

暗号化は[chezmoi](https://www.chezmoi.io/)の暗号化ファイル属性を用い、ageで暗号化したファイルとしてソースに格納する。
apply時に復号してホームへ配置する。

## 鍵の取り扱い

age鍵自体はパスフレーズで暗号化してリポジトリに収録し、ブートストラップの最初にパスフレーズを入力して復号し、所定の位置へ配置する。
以降のapplyでは、この鍵で暗号化ファイルが復号される。
鍵の復号はブートストラップで一度だけ行う。
実行順序はブートストラップの節で定める。
